# 📝 Gerenciador de Tarefas CLI

Projeto acadêmico desenvolvido para o curso da **Lions Startups (Turma da Noite)**. Trata-se de um sistema de linha de comando para gerenciar tarefas pessoais diretamente no terminal, rodando inteiramente em Node.js sem uso de bibliotecas externas complexas.

## 👥 Integrantes da Equipe
- **Henry** (Líder / Setup / Edição)
- **Wesley** (Adição de Tarefas)
- **Cabelo** (Listagem e Conclusão)

## 🚀 Como executar o projeto

1. Certifique-se de ter o Node.js instalado no seu computador.
2. Clone este repositório ou baixe os arquivos.
3. Abra o terminal na pasta raiz do projeto.
4. Digite o comando abaixo e aperte ENTER:

```bash
node index.js
